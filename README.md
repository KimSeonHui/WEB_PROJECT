# WEB_PROJECT : 메뉴얼 게시판
기존에 있던 사내 메뉴얼 게시판을 대신해 효율적이게 사용할 수 있도록 하는 게시판 구축

**목적**
- 좀 더 효율적인 메뉴얼 관리
- 메뉴얼을 빠르게 검색하기 위해
